Instala Ubuntu 20.04.1 LTS, la nueva versión de mantenimiento

Ubuntu es la distribución Linux favorita entre los usuarios que empiezan a dar los primeros pasos dentro de este sistema operativo alternativo a Windows. La distro de Canonical mantiene un equilibrio entre facilidad de uso y mantener la esencia de Linux. La última versión de esta distro que ha visto la luz del día ha sido Ubuntu 20.04 LTS, una versión muy importante que ofrecerá a los usuarios 5 años de soporte. Y, como parte de su mantenimiento, Canonical acaba de lanzar la primera actualización mayor desde el lanzamiento de esta distro el pasado mes de abril: Ubuntu 20.04.1 LTS.

Novedades de Ubuntu 20.04.1

Esta nueva versión de Ubuntu no trae grandes cambios ni novedades, sino que se centra principalmente en la corrección de errores. Gracias a esta nueva versión, los usuarios que quieran sustituir Windows por Ubuntu podrán hacerlo fácilmente, y de manera más cómoda , sin tener que instalar todas las actualizaciones y todos los parches que se han publicado desde el día de lanzamiento de esta distro. En esta actualizacion se mejoro:
  • la experiencia con el gestor de actualizaciones de Ubuntu,
  • Inclusión de nuevos drivers de NVIDIA para servidor
  • Soporte para los últimos Intel Tiger Lake
  • Microcódigo para parchear los procesadores Skylake vulnerables
Eso sí, si esperamos ver un nuevo Kernel en esta nueva versión, nos llevaremos una desilusión. Canonical no ha actualizado el Kernel de su Linux, a pesar de haber varias versiones más nuevas a las que incluye esta distro de serie.

Actualizar Ubuntu a 20.04.1

Si aún no tiene instalado Ubuntu en tu ordenador y quieres dar una oportunidad a este sistema operativo, aqui tiene nuestra guia, paso a paso Si ya esta instalado este sistema operativo en su PC, puede ponerlo al día de forma muy sencilla. Para actualizar Ubuntu a la última versión lo que debemos hacer es abrir un terminal y ejecutar en él el siguiente comando:

sudo apt update && sudo apt upgrade && sudo apt dist-upgrade

Cuando acabe el proceso tendremos la distro totalmente actualizada, al día. Ya podemos disfrutar al máximo de Linux.
